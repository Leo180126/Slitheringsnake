Chào mừng đến với tựa game Slithering Snake !!!

+   Đây là những điều cần làm sau khi bạn clone tựa game này về :>
Để bắt đầu trò chơi bạn hãy đảm bảo tải đủ các công cụ cần thiết cho Slithering snake 
như OpenCV, SFML, Cmake
Một vật màu đỏ như tờ giấy thủ công... để cầm trên tay

-   Nhấn nút run để bắt đầu trò chơi
Tựa game khởi động với 1 màn hình menu bạn hãy nhấn số 1, 2 hoặc 3 để chọn hàng tương ứng

+   Lựa chọn 1 sẽ đưa bạn vào tựa game:
Hãy cố điều khiển cho chú rắn ăn thật nhiều táo nhé
Nếu trò chơi kết thúc, bạn có thể nhấn phím Enter để chơi lại 
hoặc thoát game với Escape

+   Lựa chọn 2 hiển thị điểm số của 5 màn chơi gần nhất:
Nếu muốn thoát bạn hãy nhấn phím Q

+   Lựa chọn 3 sẽ làm bạn thoát ra khỏi tựa game :<

Trong video mình lựa chọn một môi trường nhiều noise, các bạn hãy cố chọn một nơi ổn định hơn cho việc nhận diện nhé :>
